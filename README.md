# dogapi

DogAPI is a Vue.js app that get random dogs picture from a Dog API ( https://dog.ceo/dog-api/documentation/ )

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
